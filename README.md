# Problem Statement

Recover the unknown parameters **θ**, **M**, and **X** from 1500 sampled points generated by

\[
x=t\cos\theta-e^{M|t|}\sin(0.3t)\sin\theta+X
\]

\[
y=42+t\sin\theta+e^{M|t|}\sin(0.3t)\cos\theta
\]

subject to

- \(0^\circ < \theta < 50^\circ\)
- \(-0.05 < M < 0.05\)
- \(0 < X < 100\)
- \(6 < t < 60\)

---

# Method Overview

The solution exploits the geometric structure of the given equations to reduce the optimization problem before applying numerical optimization.

### Workflow

```
Observed (x,y)
        │
        ▼
Rotation Reformulation
        │
        ▼
Recover Canonical Coordinates
        │
        ▼
Analytical Estimation of M
        │
        ▼
Differential Evolution
(Global Optimization)
        │
        ▼
Levenberg–Marquardt
(Local Refinement)
        │
        ▼
Recovered Parameters
```

### Summary

The equations are reformulated as a rigid-body rotation of a canonical curve. Using the inverse rotation, the hidden canonical coordinates are recovered analytically. The parameter **M** is estimated analytically in the transformed coordinate system, reducing the dimensionality of the optimization. Differential Evolution performs the global search, followed by Levenberg–Marquardt for local refinement.
